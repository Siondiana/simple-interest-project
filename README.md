# Simple Interest Calculator

Proyek ini berisi skrip Bash sederhana untuk menghitung bunga tunggal berdasarkan modal awal, suku bunga, dan jangka waktu yang diberikan oleh pengguna.

## Cara Menjalankan
1. Pastikan Anda memiliki akses terminal Bash.
2. Berikan izin eksekusi: `chmod +x simple-interest.sh`
3. Jalankan skrip: `./simple-interest.sh`
